## Code Metrics Spreadsheet Tool

An interactive command-line tool that displays code metrics in a spreadsheet-like format. See [SPREADSHEET.md](SPREADSHEET.md) for detailed documentation.

### Quick Start

```bash
# Build the spreadsheet tool
gcc -Wall -Wextra -o spreadsheet spreadsheet.c metrics.c

# Run it
./spreadsheet
```

For more details, see [SPREADSHEET.md](SPREADSHEET.md).
