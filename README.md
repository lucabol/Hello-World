## Visual Block Editor

This repository now includes a web-based visual block editor for creating C code!

### How to Use

1. Open `block-editor.html` in your web browser
2. Click on blocks from the left panel to add them to your program
3. Customize the values in each block
4. Click "Export to hello.c" to download your generated code
5. Compile and run: `gcc -o hello hello.c && ./hello`

See [BLOCK_EDITOR_GUIDE.md](BLOCK_EDITOR_GUIDE.md) for detailed instructions.
